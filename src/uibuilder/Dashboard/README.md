# FACIS Dashboard

The Dashboard UI for the FACIS Decentralized Catalogue Manager (DCM).

## Folders

* `/src/` - Source files served as web resources.
* `/dist/` - Distribution folder for build output.
* `/routes/` - Routing middleware definitions.
* `/api/` - REST API definitions.
* `/types/` - TypeScript definition files for IDE support.

## Files

* `package.json`: Project structure and dependencies.
* `src/index.html`: Main HTML entry point.
* `src/index.js`: UI logic.
* `src/index.css`: Custom styles.
* `tsconfig.json`: TypeScript configuration for IDE support.
