Access and Permission information
This section provides an explanation of each access.

Local Catalogue

Manage and explore your local RDF data catalogue. Upload, organize, and search through semantic assets stored directly in your system. View detailed metadata, class hierarchies, and property definitions.

Catalogue Registry

Register and manage federated catalogues from external sources. Configure SPARQL endpoints, authentication methods, and harvest strategies. Monitor connection status and coordinate distributed semantic data sources.

Schema Registry

Define and manage semantic schemas including SHACL shapes, vocabularies, and JSON-LD contexts. Create validation rules, ontologies, and standardized data models to ensure data quality and interoperability across your system.

Harvest

Execute and monitor data harvesting operations from federated catalogues. Configure harvest scopes, schedule automatic updates, and track the ingestion of semantic assets from remote sources into your local catalogue.

Admin Tools

Full administrative control over the system. Manage user accounts, assign access permissions, configure system settings, and monitor overall platform activity. Invite new users and control their access levels.
